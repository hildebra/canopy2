# canopy2

Metagenomics Canopy Clustering Implementation. 

Faster and more memory efficient implementation of the canopy clustering algorithm (doi:10.1038/nbt.2939). 



## Quickstart

```
git clone https://github.com/hildebra/canopy2.git
cd canopy2
make
./cc.bin -v
```
